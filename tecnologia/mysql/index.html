Tecnologias suportadas na Getup Cloud

Gostamos de dizer que somos poliglotas, pois oferecemos suporte a diversas linguagens e serviços através de um sistema chamado de cartridges, ou cartuchos. Os cartridges podem ser frameworks, bancos de dados, serviços de monitoramento ou qualquer outro código que sua aplicação use.

A entrega de componentes através de cartridges oferece aos desenvolvedores e administradores de sistema agilidade e padronização. Foque no código, esse é o recado!

MySQL 5.1 / 5.5

MySQL é o mais popular banco de dados relacional open source.

Simplificando a Hospedagem

Fazer o deploy de uma aplicação PHP com banco de dados MySQL a um comando de distância:

$ rhc app-create App php-5.5 mysql-5.1

O comando acima irá criar seu gear (servidor), instalar e configurar o PHP, MySQL e um repositório git no servidor.

Acesse a aplicação no endereço http://App-namespace.getup.io - acesso HTTPS também suportado e todas as aplicações possuem certificado SSL válido.

Ferramentas

Para administração via PHPMyAdmin adicione o cartridge:

$ rhc cartridge add phpmyadmin -a App

Acesse na URL https://App-namespace.getup.io/phpmyadmin

Ou se preferir administrar o banco pelo seu cliente favorito, faça um port-forward e acesse o banco via túnel SSH com total segurança.

$ rhc port-foward -a App
