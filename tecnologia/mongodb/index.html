Tecnologias Suportadas

Gostamos de dizer que somos poliglotas, pois oferecemos suporte a diversas linguagens e serviços através de um sistema chamado de cartridges, ou cartuchos. Os cartridges podem ser frameworks, bancos de dados, serviços de monitoramento ou qualquer outro código que sua aplicação use.

A entrega de componentes através de cartridges oferece aos desenvolvedores e administradores de sistema agilidade e padronização. Foque no código, esse é o recado!

MongoDB 2.4

O MongoDB é um banco de dados NoSQL orientado a documentos, utiliza o conjunto de documentos JSON.

Simplificando a Hospedagem

Fazer o deploy de uma aplicação Node.js com banco de dados MongoDB está a um comando de distância:

$ rhc app-create App nodejs-0.10 mongodb-2.4

O comando acima irá criar seu gear (servidor), instalar e configurar o Node.js, MongoDB e um repositório git no servidor.

Acesse a aplicação no endereço http://App-namespace.getup.io - acesso HTTPS também suportado e todas as aplicações possuem certificado SSL válido.

Ferramentas

Para gerenciar seu MongoDB via MMS (MongoDB Management Service) crie uma conta em https://mms.10gen.com/ e adicione o cartridge:

$ rhc cartridge add 10gen-mms-agent-0.1-a App

Se preferir, administre o banco pelo seu cliente favorito utilizando o port-forward, através de um túnel SSH com total segurança.

$ rhc port-foward -a App
