Tecnologias suportadas na Getup Cloud

Gostamos de dizer que somos poliglotas, pois oferecemos suporte a diversas linguagens e serviços através de um sistema chamado de cartridges, ou cartuchos. Os cartridges podem ser frameworks, bancos de dados, serviços de monitoramento ou qualquer outro código que sua aplicação use.

A entrega de componentes através de cartridges oferece aos desenvolvedores e administradores de sistema agilidade e padronização. Foque no código, esse é o recado!

Java

Java is a general-purpose, concurrent, class-based, object-oriented computer programming language that is specifically designed to have as few implementation dependencies as possible.

With Getup you can easily deploy and run Java applications on Tomcat (JBOSS EWS) 6 / 7 or JBoss AS7 using your favorite framework and databases. Auto-Scale to millions of users without breaking a sweat about server setup and administration. Forget about servers and focus on the code.

Simplify the lifecycle

Deploying Java application with MySQL is as easy as it can be:

$ rhc app-create App jbossews-2.0 mysql-5.1

The command will create your gear (server), install and configure Java, MySQL and git repository on the server. Access your application on the web at http://App-namespace.getup.io - HTTPS works just fine as well, and all applications have a SSL certificate.

Auto scale

What about scale? Don't worry about it, we are specialists in high traffic, simple add the -s parameter when creating your application and voilá - the platform will scale your application.

$ rhc app-create App jbossews-2.0 -s

The scalability is based on simultaneous connections. The platform will add new gears every time your application receives a traffic spike, automatically.

Frameworks, databases and other tools

Work with frameworks like Spring, Liferay and Scala/Play.

Work with database like MySQL, PostgreSQL or MongoDB through cartridges

$ rhc cartridge add mysql-5.1 -a App
